## 1. Variabili e Tipi di Dati: Etichettare il Mondo
Pensa alle **variabili** come a delle scatole su cui incolli un'etichetta. Dentro queste scatole puoi metterci un nome, un numero o un valore di verità.

*   **Stringhe (testo):** Sempre tra virgolette.
*   **Integer (numeri interi):** Senza virgola.
*   **Float (numeri decimali):** Usano il punto, non la virgola.
*   **Boolean (vero/falso):** Fondamentali per le decisioni.

```python
# Esempio Marketing
nome_campagna = "Lancio Nuova Collezione"
budget_iniziale = 1500
costo_per_click = 1.25
campagna_attiva = True

print(f"La campagna {nome_campagna} ha un budget di {budget_iniziale}€")
```

## 2. Liste e Dizionari: Organizzare l'Informazione
Nel marketing non gestisci mai un solo dato. Hai liste di influencer o schede cliente.

*   **Liste:** Un elenco ordinato di elementi.
*   **Dizionari:** Coppie di "Chiave: Valore" (come una rubrica telefonica).



```python
# Lista di influencer
influencer = ["Chiara", "Fedez", "Khaby"]

# Dizionario di un post
post_metrics = {
    "likes": 1200,
    "comments": 45,
    "shares": 10
}

print(f"Il primo influencer in lista è: {influencer[0]}")
print(f"Il post ha ricevuto {post_metrics['comments']} commenti.")
```

## 3. Controllo del Flusso (If/Else): Prendere Decisioni
Il computer deve sapere cosa fare se succede qualcosa. Ad esempio: se il budget è finito, ferma la pubblicità.



```python
budget = 50

if budget > 100:
    print("Possiamo scalare la campagna su Facebook!")
elif budget > 0:
    print("Restiamo attivi solo su Instagram.")
else:
    print("Budget esaurito. Spegnere tutto.")
```

## 4. I Cicli (For Loop): L'Automazione
Questa è la parte che ti farà risparmiare ore di lavoro. I cicli servono a ripetere un'azione per ogni elemento di una lista.



```python
clienti_premium = ["Alice", "Marco", "Giulia"]

for cliente in clienti_premium:
    print(f"Inviando email di ringraziamento a: {cliente}...")
```

## 5. Le Funzioni: Creare i Tuoi Strumenti
Una funzione è un blocco di codice che "salvi" e riusi quando vuoi, per non riscrivere sempre la stessa logica.

```python
def calcola_roi(guadagno, spesa):
    ritorno = (guadagno - spesa) / spesa
    return ritorno

risultato = calcola_roi(5000, 1000)
print(f"Il ROI della campagna è stato del {risultato * 100}%")
```
