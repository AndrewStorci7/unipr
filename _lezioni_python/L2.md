# Lezione 2: apertura di file `.csv`

## 1. Aprire e Leggere un file (Il metodo "Libro")
Immagina che un file sia un libro: per leggerlo devi aprirlo, sfogliarlo e poi ricordarti di chiuderlo. In Python usiamo `with open()` perché è "educato": apre il file e lo chiude automaticamente quando hai finito.

```python
# Leggiamo tutto il contenuto di un file di testo
with open("note_marketing.txt", "r") as file:
    contenuto = file.read()
    print(contenuto)
```

equivale a fare:

```python
file = open("note_marketing.txt", "r")
# altre istruzioni ...
file.close()
```

## 2. Leggere un CSV: La tecnica del "Taglio" (`split`)
Un file CSV è solo testo dove ogni riga è un record e i dati sono separati da una virgola (o punto e virgola). Per Python è solo una lunga stringa, ma noi possiamo "spezzettarla" usando il comando `.split()`.

```python
# Immaginiamo che il file 'dati_clienti.csv' contenga:
# Nome,Email,Città
# Maria,maria@email.com,Milano

with open("dati_clienti.csv", "r") as file:
    for riga in file:
        # .strip() toglie gli spazi vuoti e l'invio a capo alla fine
        # .split(",") divide la riga ogni volta che trova una virgola
        dati = riga.strip().split(",")
        print(f"Il cliente si chiama {dati[0]} e vive a {dati[2]}")
```

## 3. Filtrare i dati senza librerie
Supponiamo di avere un file con i risultati di diverse campagne e di voler vedere solo quelle che hanno avuto successo (ad esempio, più di 500 click).

```python
with open("report_campagne.csv", "r") as file:
    # Saltiamo l'intestazione (la prima riga con i nomi delle colonne)
    next(file) 
    
    for riga in file:
        campagna, click, costo = riga.strip().split(",")
        # Trasformiamo 'click' in numero intero per fare il confronto
        if int(click) > 500:
            print(f"Campagna Vincente: {campagna} con {click} click!")
```

## 4. Creare e Scrivere un file CSV da zero
A volte devi salvare i risultati del tuo lavoro. Usiamo la modalità `"w"` (Write). Se il file non esiste, Python lo creerà per te.

```python
nuovi_lead = [
    ["Luca", "luca@web.it"],
    ["Anna", "anna@agency.com"]
]

with open("nuovi_contatti.csv", "w") as file:
    # Scriviamo l'intestazione
    file.write("Nome,Email\n")
    
    for lead in nuovi_lead:
        # Uniamo gli elementi della lista con una virgola
        riga = ",".join(lead)
        file.write(riga + "\n")

print("File salvato correttamente!")
```

### Per riassumere:

*   **`open(nome, "r")`**: Apro per leggere.
*   **`open(nome, "w")`**: Apro per scrivere (cancella quello che c'era prima!).
*   **`.strip()`**: Pulisce la riga da spazi e invii a capo invisibili.
*   **`.split(",")`**: Trasforma una riga di testo in una **Lista** di dati pronti all'uso.
*   **`.join(lista)`**: Fa l'esatto opposto: trasforma una lista in una stringa per salvarla.
